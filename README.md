# House Pattern in C

This is a simple C pattern program that prints a house shape using stars (`*`).

The program prints:

* A triangle roof
* Two square walls side by side

## Language

* C



## How it works

* Takes number of rows from user
* Prints triangle using nested loops
* Prints two square walls using nested loops

## Example Output

## Concepts Used

* `for` loops
* nested loops
* `printf()`
